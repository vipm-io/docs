If the resolved LabVIEW target is older than 2024, the command fails fast with exit code `20` (`LABVIEW_VERSION_TOO_OLD`) and reports which LabVIEW version was selected, where the selection came from (`--labview-version`, `vipm.toml`, the `.lvproj` file, or auto-detection), and the other installed LabVIEW versions you can switch to. Install or select LabVIEW 2024 or newer. See [[LabVIEW Compatibility]](/cli/labview-interop.md)
